# Переменные используются для хранения данных
a = 5  # Присваиваем значение 5 переменной a
b = 3  # Присваиваем значение 3 переменной b

# Операторы выполняют действия над переменными
# Например, оператор "+" складывает значения
c = a + b  # Суммируем a и b, результат сохраняем в c

# Выражения состоят из переменных и операторов
# В данном случае "a + b" - это выражение

# Инструкции - это завершенные команды, которые выполняет интерпретатор
# В следующей инструкции выводим результат на экран
print("Сумма a и b:", c)  # Выводим текст и значение переменной c

# Также можно использовать другие операторы, например, вычитание
d = a - b  # Результат вычитания b из a сохраняем в d

print("Разность a и b:", d)  # Выводим результат вычитания
